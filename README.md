# Factory Salary Management

A comprehensive application to manage employee salaries, process payments, and generate payslips for a factory.

## Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.

## Deployment

This project is configured for easy deployment on [Vercel](https://vercel.com/).

1.  Push the code to a GitHub repository.
2.  Import the repository on Vercel.
3.  Vercel will automatically detect the Vite configuration and deploy the application.

## Building for Production

To create a production-ready build of the application, run:

```bash
npm run build
```

This will generate a `dist` directory with the optimized static assets. You can preview the production build locally with:

```bash
npm run preview
```
